Act as an expert software engineer and fitness data architect. Generate a highly functional, interactive, and mobile-friendly workout tracking application for a 5-day "PPL + UL" (Push, Pull, Legs, Upper, Lower) training split. 

The application must strictly map exercises to their specific anatomical muscle heads to provide real-time recovery data and tracking.

### 1. APPLICATION STRUCTURAL FRAMEWORK

Implement the following 5-Day Workout Architecture into the tracker view:

#### DAY 1: PUSH DAY (Anterior & Extension Focus)
Sequence: Heavy Compound → Stretch-Emphasized → Isolation

*   Exercise 1: Barbell Flat Bench Press [Targets: Sternocostal Chest, Lateral Triceps]
*   Exercise 2: Incline Dumbbell Press [Targets: Clavicular Chest]
*   Exercise 3: Standing Cable Lateral Raise [Targets: Lateral Deltoid]
*   Exercise 4: Seated Dumbbell Shoulder Press [Targets: Anterior Deltoid]
*   Exercise 5: High to Low Cable Flye [Targets: Lower Pec Fibers]
*   Exercise 6: Cable Tricep Extension [Targets: Triceps Long Head]
*   Exercise 7: Cable Triceps Pushdowns [Targets: Triceps Lateral & Medial Heads]

#### DAY 2: PULL DAY (Posterior & Flexion Focus)
Sequence: Width → Thickness → Unilateral → Rear Delts → Biceps Length → Biceps Peak

*   Exercise 1: Wide-Grip Lat Pulldown [Targets: Thoracic Lats]
*   Exercise 2: Chest Supported Dumbbell Row [Targets: Rhomboids & Mid-Traps]
*   Exercise 3: Single-Arm Neutral Grip Cable Row [Targets: Iliac/Lumbar Lats]
*   Exercise 4: Rear Delt Cable Flyes [Targets: Posterior Deltoid]
*   Exercise 5: Bayesian Curl [Targets: Biceps Long Head]
*   Exercise 6: Preacher Curl [Targets: Biceps Short Head]
*   Exercise 7: Reverse Grip Curl [Targets: Brachioradialis & Brachialis]

#### DAY 3: LEGS & ABS (High-Efficiency Lower Body)
Sequence: Pre-Exhaust Quads → Compound → Hinge/Glute → Hamstring Isolation → Calves → Abs

*   Exercise 1: Seated Leg Extensions [Targets: Rectus Femoris Pre-Exhaust]
*   Exercise 2: Barbell Squat [Targets: Total Quads & Glutes]
*   Exercise 3: Romanian Deadlift (Week A) OR Barbell Hip Thrust (Week B) [Hamstrings/Glutes Alternate]
*   Exercise 4: Seated Leg Curl [Targets: Hamstring Knee Flexion]
*   Exercise 5: Standing Calf Raises [Targets: Gastrocnemius]
*   Exercise 6: Weighted Rope Cable Crunches [Targets: Upper Rectus Abdominis]
*   Exercise 7: Cable Woodchopper [Targets: Obliques & Transversus Abdominis]

#### DAY 4: UPPER DAY (Antagonist Compound Duels)
Sequence: Horizontal Duel → Vertical Duel → Isolation Finishers

**Duel 1 (Horizontal Plane)**
*   Exercise 1: Flat Dumbbell Bench Press [Targets: Sternocostal Chest]
*   Exercise 2: Chest-Supported Dumbbell Row [Targets: Rhomboids & Mid-Traps]

**Duel 2 (Vertical Plane)**
*   Exercise 3: Seated Dumbbell or Smith Machine Shoulder Press [Targets: Anterior Deltoid]
*   Exercise 4: Close-Grip Neutral Lat Pulldown [Targets: Lower Lats]

**The Finishers**
*   Exercise 5: Cable Lateral Raise [Targets: Lateral Deltoid]
*   Exercise 6: Cable Bicep Curls [Targets: Total Biceps]
*   Exercise 7: Cable Triceps Pushdowns [Targets: Triceps Lateral & Medial Heads]

#### DAY 5: LOWER DAY & ABS (Heavy Axial Loading)
Sequence: Heavy Squat → Accessory Press → Heavy Hinge → Isolations → Calves → Abs

*   Exercise 1: Barbell Back Squat [Targets: Total Quads & Glutes — Strength Focus]
*   Exercise 2: Leg Press (Full Depth) [Targets: Quad Overload]
*   Exercise 3: Romanian Deadlift [Targets: Total Hamstrings]
*   Exercise 4: Seated Leg Curl [Targets: Hamstring Knee Flexion]
*   Exercise 5: Seated Calf Raise [Targets: Soleus]
*   Exercise 6: Hanging Leg Raises [Targets: Lower Rectus Abdominis & Transversus Abdominis]
*   Exercise 7: Cable Crunches [Targets: Upper Rectus Abdominis]
*   Exercise 8: Cable Woodchopper [Targets: Obliques & Transversus Abdominis]

### 2. DATA INPUTS & FUNCTIONAL REQUIREMENTS

For EACH exercise listed above, provide the following interactive tracking states:
1.  **Set Logger:** Rows for Set Number, Weight Input (lbs/kg), and Reps Completed Input.
2.  **RPE Selector:** A dropdown menu from 1 to 10 to measure the Rate of Perceived Exertion (Intensity).
3.  **Rest Timer:** An in-app countdown timer initialized to the optimal rest time specified for each movement.
4.  **1RM Calculator:** Automate a background calculation using Epley’s formula: 1RM = Weight * (1 + Reps / 30) to log historical performance.

### 3. ADVANCED ANALYTICS (BIOLOGICAL INSIGHTS)

Build a data visualization or metric dashboard that processes logged sessions to show:
*   **Volume Tracking:** Total active working sets per muscle group per week.
*   **Anatomical Recovery Score:** When Day 1 (Push) is completed, tag Chest and Triceps as "Fatigued - 48h timer active". Visual flags should show these muscle systems changing color dynamically from Red (Fatigued) to Green (Recovered) across the schedule.

Deliver the complete layout code (e.g., HTML/JS/Tailwind CSS stack, Python Streamlit, or a clean UI component framework layout) ready to deploy.
